RGPD - Regulamento Geral de Protecção de Dados

O Regulamento Geral de Proteção de Dados vem introduzir novas obrigações a quem faz processamento de dados pessoais. As obrigações estão dependentes de tipo de dados que sejam recolhidos e como.

Requisitos WooCommerce

  • PHP 5.6 ou superior (recomendado)
  • MySQL 5.6 ou superior (recomendado)
  • Limite de memória no WordPress de 64mb ou superior. 128mb são preferiveis.

NÃO USEM VERSÕES DESATUALIZADAS

Usem sempre SSL!

SSL - Certificados Digitais

  • Previne roubos de dados, pois são transferidos encriptados
  • Confiança ao cliente
  • Exigido por entidades de cartão de crédito
  • Let’s encrypt - certificados SSL gratuitos, renovados a cada 90 dias.
