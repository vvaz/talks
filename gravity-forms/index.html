Fields de administração

Campos que recolhem dados, mas os utilizadores não os veem.

Exemplos: Podemos colocar um "hidden field" para fazer tracking de onde o utilizador chegou ao form atual.

Conditional Logic

Mostrar ou esconder campos baseado em respostas anteriores.

A lógica condicional também funciona com os admin fields.

Os campos de lógica condicional, podem também guiar os utilizadores que estão a responder ao vosso formulário, a não cometerem erros

Podem ser usados para esconder campos até que seja realmente necessário a resposta do utilizador.

Registo de Eventos

  • Categorias de inscrição com vários preços
  • Menores de idade têm de carregar aprovação de encarregado de educação
  • Maiores de 18 têm de ter possibilidade de adicionar crianças para as refeições servidas durante o evento
  • Participantes podem escolher alojamento e influencia no preço final
  • Envio de notificação para admin e para inscrito
