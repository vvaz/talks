.com ou .org?
São distintos!

.com

WordPress.com é uma plataforma onde podem ter o vosso site gratuitamente, mas com algumas limitações.

Podem até fazer um upgrade para um plano pago, tendo mais espaço em disco por exemplo.

Anúncios no .com
Limitações incluem:
  • Limitado aos temas disponíveis no wordpress.com
  • Sem plugins próprios
  • Limitação de espaço em disco
  • Não há acesso FTP
  • Limitação no conteúdo
.org Podem fazem o download gratuitamente e alojar num servidor. Vai precisar de comprar um alojamento. Completo controlo do seu site / conteúdo.

Temas

Os temas de WordPress vivem num subdirectorio muito especifico que não pode ser movido (wp-content/themes/). O subdirectório do tema contém todos os ficheiros de estilo, ficheiros template, ficheiro de funções (functions.php), imagens e ficheiros JavaScript

Por exemplo, um tema chamado "teste" iria estar no directório wp-content/themes/test/.

O WordPress traz sempre um tema default em cada instalação. Um bom exercício é analisar os ficheiros deste tema para ter uma melhor ideia da estrutura.

Os temas de WordPress são constituidos, normalmente, por três ficheiros principais em adição a imagens e ficheiros de JavaScript.

  1. O ficheiro de estilos chamado style.css, que controla a apresentação (design e layout) das páginas do site.
  2. WordPress template files que controlam a forma como as páginas do site geram a informação contida na base de dados para ser apresentada no site.
  3. O ficheiro opcional de funções(functions.php).

No minimo um tema de WordPress é constituido por dois ficheiros:

  • style.css
  • index.php

Este ficheiros vão para dentro da pasta do tema. O index.php é muito flexivel. Pode ser usado para chamar todos os outros ficheiros como o header, sidebar, footer, content, categories, archives, search, error, e qualquer outra página criada.

Ou então pode ser divido em ficheiros template. Se não fornecermos um ficheiro template ao tema, o WordPress poderá ter ficheiros standard ou funções para executar esse trabalho. Por exemplo, se não criarmos o ficheiro searchform.php, o Wordpress tem uma função "default" que irá mostrar uma caixa de pesquisa.

Ficheiros que normalmente encontramos num tema:

  • comments.php
  • comments-popup.php
  • footer.php
  • header.php
  • sidebar.php

Usando estes ficheiros podemos depois inclui-los no index.php para os fazer aparecer onde queremos na página.

  • Para incluir o header, usamos get_header()
  • Para incluir a sidebar, usamos get_sidebar()
  • Para incluir o footer, usamos get_footer()
  • Para incluir o search form, usamos get_search_form()

Um exemplo de como chamar estes ficheiros no index.php ou outra página seria usar o seguinte código:


        get_header();
get_footer();

Plugins

Os Plugins possibilitam a fácil modificação, costumização e melhoramento de um site WordPress.

Os plugins disponíveis no repositório do WordPress são criados por voluntários e, normalmente, são gratuitos. Estes plugins alojados no directório do WordPress são considerados seguros e testados.

Uma nota importante é que estes plugins são da responsabilidade do autor e do utilizador e, tipicamente, estão em constante desenvolvimento.

Para pesquisar plugins basta ir ao dashboard do WordPress, no ecrã Plugins > Add New.
