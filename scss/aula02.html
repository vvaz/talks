Mixins

Os mixins são usados para substituir declarações inteiras de CSS. Podemos guardar num mixin e reutilizar estilos de CSS pelos nossos ficheiros todos.

O código é aplicado através do @include

Tem por boas práticas residir num ficheiro externo tal como as variáveis

 
          
              @mixin alert {
                background-color: #fff;
                color: #000;
              }
              .error {
                @include alert;
              }

Mixin Arguments

Podemos também passar argumentos aos Mixins, dando uso então às nossas variáveis:

            
                @mixing alert-text($color) {
                  background-color: $color;
                  color: #000;
                }
                .error {
                  @include alert(blue);
                }
