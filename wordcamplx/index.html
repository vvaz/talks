ARQUITETURA DE INFORMAÇÃO

"Arquitetura de informação é a categorização de informação numa estrutura coerente, preferencialmente uma que o utilizador compreenda rapidamente (ou até intuitivamente)"

Artigos

Têm um editor visual, titulo e podem ter imagens

Têm categorias e/ou tags.

Páginas

Podem ter hierarquia.

Ao contrário dos artigos, as páginas não têm categorias ou tags.

Páginas e Artigos

Ambos têm um editor de conteúdo, podemos definir uma imagem de destaque e podem ter comentários.

A nossa Home Page pode ser uma listagem dos últimos artigos ou uma página estática.

Taxonomias

Categorias e Tags

Categorias

São excelentes para organizar conteúdo, podem ter hierarquia e são obrigatórias.

Tags

São utilizadas para descrever conteúdo, são lineares (ou seja, não têm hierarquia) e são opcionais.

Menus

Suportam todo o tipo de contúdo desde categorias, tags ou links personalizados.

Têm um estrutura hierárquica que podemos modificar através de um sistema "Drag-and-Drop".

Estão dependentes do tema que se usa.

Permalinks

É a estrutura de URL's do site.

Permitem a criação de uma estrutura de URL's personalizada.

Muito importante para SEO.

Opções

Perfis de Utilizadores

Esta é uma secção importante que deve ser gerida com cuidado. De raiz temos 5 tipos de utilizadores:

  • Administrador: Tem acesso a todas as funcionalidades e opcções.
  • Editor: Pode criar e eliminar páginas, gerir e publicar artigos e carregar imagens e outros ficheiros. Pode ainda rever e publicar artigos submetidos por outros.
  • Autor: Pode escrever e publicar os seus artigos, incluindo carregar e inserir imagens.
  • Colaborador: Pode escrever artigos, mas não os pode publicar. Têm de ser submetidos para revisão.
  • Subscritor: Pode gerir o seu perfil, ler o que publicam e deixar comentários.

Temas

Os temas de WordPress vivem num subdirectorio muito especifico que não pode ser movido (wp-content/themes/). O subdirectório do tema contém todos os ficheiros de estilo, ficheiros template,ficheiro de funções(functions.php), imagens e ficheiros JavaScript

Por exemplo, um tema chamado "teste" iria estar no directório wp-content/themes/teste/.

O WordPress traz sempre um tema default em cada instalação. Um bom exercício é analisar os ficheiros deste tema para ter uma melhor ideia da estrutura.

Os temas de WordPress são constituidos, normalmente, por três ficheiros principais em adição a imagens e ficheiros de JavaScript.

  1. O ficheiro de estilos chamado style.css, que controla a apresentação (design e layout) das páginas do site.
  2. WordPress template files que controlam a forma como as páginas do site geram a informação contida na base de dados para ser apresentada no site.
  3. O ficheiro opcional de funções(functions.php).

No minimo um tema de WordPress é constituido por dois ficheiros:

  • style.css
  • index.php
