Media Queries

  • handheld - para dispositivos de mão
  • print - para impressão em papel
  • projection - para apresentações como powerpoint
  • screen - monitores desktop ou ecrans de alta resolução
  • tv - para dispositivos com pouca resolução
  • aural - sintetizadores de braille
  • all - para todos os dispositivos

Onde temos a internet em 2019?
As Media Queries surgiram com o CSS3 e é destinado a detectar as caracteristicas do dispositivo e do browser do utilizador.
A Media Query faz perguntas ao browser e a sua resposta é sempre verdadeira ou falsa.
Permite ver parâmetros como a altura e largura do browser, resolução e orientação do dispositivo
<link rel="stylesheet" href="estilos.css" media="screen" />

ou:

@media print {}

Principais resoluções utilizadas:

  • 320px - smartphones no modo retrato
  • 480px - smartphones em modo paisagem
  • 600px - tablets pequenos
  • 768px - tablets com maior resolução (ex: iPad)
  • 1200px - monitores widescreen para desktop

Existem operadores em Media Queries:

Not, And e Only

Operadores lógicos:

NOT 

					 		
							<link rel="stylesheet" href="estilo.css" media="all and (not color)" />

Only 

					 		
							<link rel="stylesheet" href="estilo.css" media="only screen and (color)" />

OR 

					 		
							<link rel="stylesheet" href="estilo.css" media="screen and (color), projection and (color)" />

Fluid vs Responsive

Os layouts fluid, fluem consoante o dispositivo. Ex: tabelas em percentagem

Layout responsive, responde e adapta-se ao ecra do dispositivo. (ex: colunas que passam para uma linha diferente)
