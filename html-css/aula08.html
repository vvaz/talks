Navbars - elementos de navegação

É uma "Lista de links" onde o visitante do nosso website pode clicar para navegar no nosso projeto.

Figma

Requisitos do exercício:

  • Tem vários "estados" e temos de os fazer a todos, podem ser no mesmo ficheiro ou em ficheiros diferentes
  • Vamos usar uma library de CSS chamada FontAwesome
  • Temos de usar Google Fonts
  • Temos de usar o reset.css do Eric Meyer
