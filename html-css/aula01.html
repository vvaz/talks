O que o cliente (com um browser) faz
O HTML não é uma linguagem de programação, nada mais é do que texto.

O HTML e o CSS não se executam, logo não podem ser considerados uma linguagem de programação.

Ao contrário do JavaScript que se pode executar há diferenças. No HTML e CSS apenas definimos como o browser deve interpretar elementos.

HTML - Hyper Text Markup Language

CSS - Cascading Style Sheets

Utilizando a analogia de uma casa, o HTML descreve os materiais usados para fazer a casa e o que está na casa, o CSS descreve as cores e posições dos elementos.

1. O HTML é a nossa estrutura, vai conter todo o texto e imagens, e normalmente vai agrupar coisas.

2. O CSS é a nossa folha de estilos, regras a seguir de como o conteúdo deve aparecer, que cor usar, que tamanho, que fonte,etc.

3. O frontend (normalmente em conjunto com o JavaScript), é o nosso elemento visual, o que é produzido pelo HTML e CSS. Com o JavaScript podemos também tornar o HTML e CSS reactivo.

4. O backend (que pode também usar JavaScript ou outra linguagem), é o sitio que recebe pedidos e retorna a devida resposta.

W3C - World Wide Web Consortium

É uma comunidade internacional, onde os seus membros associados, empregados e o publico em geral, trabalham em conjunto para definir os Web Standards.

A declaração representa o tipo de documento e ajuda os browsers a mostrar correctamente as páginas web.

Antigas declarações de documento

HTML 4.01 Transitional

						 <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

						 <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

HTML5 vs XHTML

o XHTML, criado em 2000, é mais rigido na forma como se escreve não admitindo erros.

exemplo:

						 <p><strong>Conteúdo a negrito</p></strong>

Pode ser usado no HTML5 mas não no XHTML

A sintaxe do HTML baseia-se nas tags.

Tags são rótulos usados para informar ao browser como o conteúdo deverá ser renderizado ou apresentado.

Tags são o conjunto de caracteres que formam um elemento, ou seja, quando nos referenciamos à Tag "p" estamos a falar de <p>

Existem dois tipos de Tags, as que necessitam de fecho e as que não necessitam de fecho.

<tag> ...... </tag> O browser irá processar tudo o que estiver contido entre a tag de inicio <> e a tag de fim
Algumas tags, as chamadas de comandos isolados, não precisam de ter conteúdo entre o inicio e o fim. Por exemplo: <br /> ou <img />

Atributos de uma tag

Todas as tags HTML podem ter atributos, que o browser pode interpretar. Por exemplo, a tag <a> tem um atributo href, que o browser interpreta como o destino de uma hiperligação.

						
						<a href="https://google.com">Visitar o Google </a>

Por fim, o conjunto de tags, atributos e o seu conteúdo, chama-se de elemento.

Um elemento consiste em:

						
							<tagname>O conteúdo a ser mostrado... </tagname>

Recomendações

Usar sempre tags escritas em lowercase.

O standard em HTML5 não requer que as tags sejam escrita em letra minuscula, mas o W3C recomenda. Já no XHTML é obrigatório.

Com a evolução da web, foram surgindo tags standard, usadas na maior parte dos casos, outras tags apareceram para substituir tags antigas.

Atributos

href

Os links são definidos pela tag <a> e os endereços são especificados no atributo href 

						
							<a href="https://google.pt">Visitar o Google</a>

Atributos

src

As imagens são definidas pela tag <img> e a localização e nome do ficheiro são especificados pelo atributo src 

						
							<img src="img_001.jpg" />

Atributos

width e height

As imagens poderão ter atributos de tamanho, definidos por width (largura) e height (altura). 

						
							<img src="img_001.jpg" width="500" height="600" />

Não precisamos de especificar a unidade pixel, ou px.

Atributos

alt

O atributo alt, especifica um texto alternativo a ser usado, quando a imagem não pode ser apresentada.
Este atributo pode ser lido por "screen readers". Deste modo, uma pessoa portadora de deficiência visual que "ouça" a página, percebe que existe uma imagem na página. 

						
							<img src="img_001.jpg" alt="Uma pessoa a andar de bicicleta" />

Atributos

style

O style é utiliazdo para estilizar o elemento, como a sua cor, font utilizada, tamanho, etc. 

						
							<p style="color:blue">Conteúdo de texto a azul </p>

Atributos

lang

A linguagem do documento pode ser declarada na tag <html> 

						
							<html lang="pt-PT">

Declarar a linguagem é importante tanto para aplicações de acessibilidade, como para os motores de pesquisa identificarem a lingua do website.

Atributos

title

O atributo title, apresenta uma pequena caixa de texto quando o utilizador coloca o rato sobre um elemento HTML. 

						
							<p title="sou uma tooltip">Isto é um parágrafo de texto. </p>
