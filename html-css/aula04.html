Uma ferramenta indispensável na criação de páginas web são os estilos CSS - Cascading Style Sheets.

CSS é uma linguagem de estilos que descreve como o HTML deve ser interpretado e utilizado pelo browser.

O CSS3 é a última revisão aos estilos que se podem utilizar.

Alguns dos novos estilos de CSS3 incluem dropshadows (sombreados), cantos redondos, animações, transparências e muito mais.

CSS tem por definição Cascading Style Sheet e é usado para estilizar elementos escritos numa linguagem de marcação como HTML O CSS separa o conteúdo da representação visual do site. Utilizando o CSS é possível alterar a cor do texto e do fundo, fonte e espaçamento entre parágrafos

Também pode criar tabelas, usar variações de layouts, ajustar imagens para cada ecrã e assim por diante

Antes de usar CSS, toda a estilização tinha que ser incluída no código HTML. Significa que tínhamos que descrever separadamente todo o background, as cores das fontes, os alinhamentos, etc.

Mas o CSS permite estilizar tudo num ficheiro diferente, criando assim o estilo separadamente. E, mais tarde, fazer a integração do arquivo CSS na parte superior do código HTML. Isto mantém o HTML limpo e fácil de manter

Estilos CSS: Interno

Estilos CSS feitos desta forma são carregados cada vez que um site é atualizado, o que pode aumentar o tempo de carregamento. No entanto o documento HTML vai aumentar muito o tamanho e não podemos colocar nada na "memória cache".

Coloca-se no head do documento

Estilos CSS: Externo

O método EXTERNO pode ser o mais conveniente. Tudo é feito externamente num ficheiro .css. Isso significa que podemos fazer todo o estilo num ficheiro separado e aplicar o CSS a qualquer página desejada. O estilo externo também pode melhorar o tempo de carregamento e pode ser colocado em cache.

Estilos CSS: Inline

Trabalha com elementos específicos que possuem o atributo style para cada tag HTML. Cada componente deve ser estilizado, por isso talvez não seja o melhor ou o mais rápido para lidar com CSS, mas pode ser útil, por exemplo, para alterar um único elemento, visualizar rápidamente as alterações ou se não tiver acesso aos ficheiros CSS.

Diferentes formas de adicionar estilos CSS a um elemento HTML.

Como atributo style="" dentro de um elemento HTML
Utilizando classes ou ids nos elementos HTML e organizando o código CSS dentro da tag <head>

Nota: As classes podem repetir-se em todo o código HTML, mas as IDs são únicas ao elemento.

Ou (a melhor forma) utilizando um ficheiro externo de CSS
<link rel="stylesheet" type="text/css" href="estilos.css">
Hierarquias de CSS
  • !important
  • Estilos no elemento HTML inseridos no atributo
  • Estilos em ficheiros externos
  • Estilos por defeito no browser

Os estilos CSS3 incluem parametros técnicos que os designers estão acustomados a utilizar nos seus trabalhos, e finalmente podem ser integrados numa página web, dentro de standards para os browsers seguirem.
