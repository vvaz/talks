As divs

						
							<div>
								-- outros elementos HTML aqui --
							</div>

DIVs - division. É uma espécie de uma mala de viagem, onde temos todos os nossos pertences lá dentro. Serve para agrupar. É melhor definido pelo seu conteúdo. Vamos usar muito esta tag, é extremamente essencial para CSS.

A div em condições normais força uma quebra de linha para o elemento que vem depois da div.

As spans

						
						temos algum texto <span> que queremos destacar </span>

span. Tal como a div serve para agrupar, mas apenas texto. Usado quando queremos destacar algo no nosso conteúdo. Não força uma quebra de linha

Formulários

A experiência do utilizador num website não é só carregar em links e mudar de páginas. Às vezes o utilizador terá que interagir com o website para realizar diferentes tarefas.

  • Efectuar autenticação numa área privada (login)
  • Introduzir informação pessoal, como nome, morada, dados de cartão de crédito, etc
  • Filtragem de conteúdo (por exemplo com checkboxes e dropwdowns)
  • Realizar uma pesquisa no website
  • Carregar ficheiros

Existem várias tags HTML que nos ajudam a criar formulários nas nossas páginas web

  • Inputs de texto
  • Radio Buttons
  • Checkboxes
  • Dropdowns
  • Upload widgets
  • Submit buttons

Muitas destes elementos, são atributos da tag <input>

O elemento Form

A tag <form> constitui um elemento que define uma parte interactiva numa página. Todas as tags de controlo (exemplo <input>, <textarea>, <button>), têm de estar inseridas dentro da tag <form>

exemplo: 
					 	
							<form action="/send">
								<input type="text" />
							</form>

Os dados são então enviados para o servidor, que atrávés de uma linguagem de programação os processa. Isto é definido pelo atributo action, que nós vamos sempre colocar action="/send"

Como estes dados são processados vai para além do HTML e CSS

					 
					<form action="/send"> ... </form>

					<input> - elemento de entrada

Elementos de entrada podem conter checkboxes (escolha multipla), campos de password, radio buttons (escolha singular), botão de envio (submit) e é definido pelo atributo type=""

					 
						<input type="text">	Define uma linha de texto
						<input type="radio">	Define um radio button - escolha uma opção entre várias
						<input type="submit">	Gera um botão de envio de dados
					
				
					 
						<form action="/send">
						Primeiro nome:

						<input type="text" name="primeironome" /<

						Último nome:

						<input type="text" name="ultimonome" />
						</form>
					
				
					 
						<form action="/send">
					  	<input type="radio" name="gender" value="masculino" checked> Masculino

					  	<input type="radio" name="gender" value="feminino"> Feminino

					  	<input type="radio" name="gender" value="outro"> Outro
						</form>

Atributo style

O atributo style pode ser usado na maior parte das tags, e deixa-nos adicionar estilos ao elemento HTML.

Alguns dos estilos que vamos utilizar são: background-color, color, font-size, width, height, margin e padding

exemplo: 
					 

					<div style="background-color: red; width: 300px;">
					  Conteúdo
					</div>
