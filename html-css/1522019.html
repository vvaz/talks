Descobrimos a tag <video> e <audio> e como as conseguimos utilizar para incorporar ficheiros no nosso servidor 

					 
						<video width="320" height="240" controls>
						  <source src="SampleVideo_1280x720_2mb.mp4" type="video/mp4">
						  <source src="SampleVideo_1280x720_2mb.ogg" type="video/mp4">
						  O seu browser não suporta a tag video
						</video>
					
				
					 
						<audio controls>
							<source src="SampleAudio_0.4mb.mp4" type="video/mp4">
							<source src="SampleAudio_0.4mb.ogg" type="video/mp4">
							O seu browser não suporta a tag audio.
						</video>

O Atributo "controls" adiciona os controlos de play, pause e volume

O elemento "source" deixa-nos adicionar vários ficheiros com terminações diferentes e o browser irá reproduzir o primeiro ficheiro suportado.

O texto apenas será mostrado se o browser não suportar a tag audio ou video.

Diferentes formas de adicionar estilos CSS a um elemento HTML.

Como atributo style="" dentro de um elemento HTML
Utilizando classes ou ids nos elementos HTML e organizando o código CSS dentro da tag <head>

Nota: As classes podem repetir-se em todo o código HTML, mas as IDs são únicas ao elemento.

Ou (a melhor forma) utilizando um ficheiro externo de CSS
<link rel="stylesheet" type="text/css" href="estilos.css">

Uma ferramenta indispensável na criação de páginas web são os estilos CSS - Cascading Style Sheets.

CSS é uma linguagem de estilos que descreve como o HTML deve ser interpretado e utilizado pelo browser.

O CSS3 é a última revisão aos estilos que se podem utilizar.

Algumas dos novos estilos de CSS3 incluem dropshadows (sombreados), cantos redondos, animações, transparencias e muito mais.

Os estilos CSS3 incluem parametros técnicos que os designers estão acustomados a utilizar nos seus trabalhos, e finalmente podem ser integrados numa página web, dentro de standards para os browsers seguirem.

Um exemplo de como os CSS3 evoluiram está nos background em forma de gradientes. 

					 
						<div style="background-image: url(gradient1,png);
						 background-repeat: repeat-x;">	</div>
					
				
					 
						<div style="linear-gradient(to bottom, #8E0E00 0%,
						#1F1C18 100%);">	</div>
