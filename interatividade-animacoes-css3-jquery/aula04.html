HTML

É uma linguagem que disponibiliza conteúdos que o browser interpreta e os mostra. Utilizado para fazer páginas web.

CSS

É um documento com propriedade de estilos para personalizar o design do nosso HTML.

JavaScript

É uma linguagem de programação que adiciona interactividade às nossas páginas web.

jQuery

É uma library (biblioteca) em JavaScript e foi desenvolvida para facilitar a tarefa dos programadores

jQuery

  • Agrupa tarefas comuns que requerem muitas linhas de código em JavaScript em metodos que podemos chamar com uma linha de código.
  • Simplifica acções complicadas como pedidos AJAX e a manipuação do DOM

Funções:

  • Manipuação do DOM
  • Manipuação do CSS
  • Eventos em HTML
  • Efeitos e Animações
  • AJAX requests
  • Utilities

Temos de ter algum conhecimento em JavaScript para conseguir usar o jQuery. Não se foquem só em jQuery, JavaScript é essencial.

Evitando conflitos

Os conflitos começam a aparecer quando começamos a usar mais do que uma library que utilize o:

                
                    $();

Uma das libraries que dá conflicto com jQuery é a Prototype

Podemos ultrapassar isso criando uma variável diferente para usar o jQuery

                
                    var $j = jQuery.noConflict();
                    $j();
                    $j(document).ready(function() {
                        alert("assim funciona");
                    })

Usando nativamente:

                
                    jQuery(document).ready(function() {});

Como fazemos para verificar se existe algum elemento no DOM com uma class?

jquery4.html 
                
                    if(condição) {
                        // código a executar se for verdadeiro
                    } 
                    else {
                        // código a executar se for falso
                    }
                
            
                
                    if( $('.class') ) {
                        // codigo
                    }

Isto vai dar sempre falso, porque o selector retorna sempre alguma coisa.

                
                    if( $('.class').length > 0 ) {
                        // codigo
                    }

Assim, se o length do objecto for maior que 0, é sinal que existe.

Exercicio 1 jQuery

Lendo dados guardados nos atributos com jQuery

                
                    $j(document).ready(function() {
                        $('a').attr('border');
                        $('a').attr('href');
                    });

Mudando atributos com jQuery

                
                    $j(document).ready(function() {
                        $('a').attr('href','https://google.com');
                    });

Mudando conteúdo com jQuery

                
                    $j(document).ready(function() {
                        $('.class').html("Alguma coisa diferente");
                    });

Mudando conteúdo com jQuery

                
                    $j(document).ready(function() {
                        $('.class').text('Alguma coisa diferente');
                    });

Adicionando classes com jQuery

                
                    $j(document).ready(function() {
                        $(element).addClass('novaClass');
                    });

Outras acoes de jQuery que nos vão ser preciosas:

                
                    $j(document).ready(function() {
                        $(element).find('img');
                        $(element).find('img').width(200);
                        $(element).find('img').height(200);
                    });

Outras ações de jQuery que nos vão ser preciosas:

                
                    $j(document).ready(function() {
                        $(element).after(''); // Insere código depois do elemento
                        $(element).before(''); // Insere código antes do elemento
                        $(element).append(''); // Insere código dentro do elemento
                        $(element).prepend(''); // Insere código dentro do elemento
                    });
