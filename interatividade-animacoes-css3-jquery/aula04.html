HTML

É uma linguagem que disponibiliza conteúdos que o browser interpreta e os mostra. Utilizado para fazer páginas web.

CSS

É um documento com propriedade de estilos para personalizar o design do nosso HTML.

JavaScript

É uma linguagem de programação que adiciona interactividade às nossas páginas web.

jQuery

É uma library (biblioteca) em JavaScript e foi desenvolvida para facilitar a tarefa dos programadores

jQuery

  • Agrupa tarefas comuns que requerem muitas linhas de código em JavaScript em metodos que podemos chamar com uma linha de código.
  • Simplifica acções complicadas como pedidos AJAX e a manipuação do DOM

Funções:

  • Manipuação do DOM
  • Manipuação do CSS
  • Eventos em HTML
  • Efeitos e Animações
  • AJAX requests
  • Utilities

Temos de ter algum conhecimento em JavaScript para conseguir usar o jQuery. Não se foquem só em jQuery, JavaScript é essencial.
