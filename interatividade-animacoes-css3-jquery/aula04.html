HTML

CSS

JavaScript

É uma linguagem de programação que adiciona interactividade às nossas páginas web.

jQuery

É uma library (biblioteca) em JavaScript e foi desenvolvida para facilitar a tarefa dos programadores

jQuery

  • Agrupa tarefas comuns que requerem muitas linhas de código em JavaScript em métodos que podemos chamar com uma linha de código.
  • Simplifica ações complicadas como pedidos AJAX e a manipuação do DOM

Funções:

  • Manipulação do DOM
  • Manipulação do CSS
  • Eventos em HTML
  • Efeitos e Animações
  • AJAX requests
  • Utilities

Evitando conflitos

Os conflitos começam a aparecer quando começamos a usar mais do que uma library que utilize o:

                
                    $();

Uma das libraries que dá conflicto com jQuery é a Prototype

Podemos ultrapassar isso criando uma variável diferente para usar o jQuery

                
                    var $j = jQuery.noConflict();
                    $j();
                    $j(document).ready(function() {
                        alert("assim funciona");
                    })

Usando nativamente:

                
                    jQuery(document).ready(function() {});

                        $('.class').html("Alguma coisa diferente");
                    });

Mudando conteúdo com jQuery

                
                    $j(document).ready(function() {
                        $('.class').text('Alguma coisa diferente');
                    });

Adicionando classes com jQuery

                
                    $j(document).ready(function() {
                        $(element).addClass('novaClass');
                    });

Outras acoes de jQuery que nos vão ser preciosas:

                
                    $j(document).ready(function() {
                        $(element).find('img');
                        $(element).find('img').width(200);
                        $(element).find('img').height(200);
                    });

Outras ações de jQuery que nos vão ser preciosas:

                
                    $j(document).ready(function() {
                        $(element).after(''); // Insere código depois do elemento
                        $(element).before(''); // Insere código antes do elemento
                        $(element).append(''); // Insere código dentro do elemento
                        $(element).prepend(''); // Insere código dentro do elemento
                    });

O .append() vai adicionar coisas ao que já existe, e o .html vai substituir todo o código que se encontra nesse elemento.

Eventos:

Todas as acções de utilizador, que uma página web pode responder, chamam-se eventos.

Exemplos de eventos:

  • click
  • hover
  • mouseenter
  • mouseleave
  • keypress
  • keydown
  • keyup

Exemplos de eventos:

  • submit
  • change
  • focus
  • load
  • resize
  • scroll

Sintaxe:

                

                    $("p").click(); 
                    
                    // o que acontece depois de um evento:
                    $("p").click(function(){
                        // action goes here!!
                    });

Eventos - click

Ver documentação aqui - jQuery Doc

                
                    $(elemento).click(function() {
                        alert( "o elemento foi clicado" );
                      });

O Método on() permite-nos ter vários eventos:

                
                    // um evento
                    $("p").on("click", function(){
                        $(this).hide();
                    });

O Método on() permite-nos ter vários eventos:

                
                    // vários eventos:
                    $("p").on({
                        mouseenter: function(){
                          $(this).css("background-color", "lightgray");
                        },
                        mouseleave: function(){
                          $(this).css("background-color", "lightblue");
                        },
                        click: function(){
                          $(this).css("background-color", "yellow");
                        }
                      });

Efeitos

                
                    $j(document).ready(function() {
                       $(elemento).hide(speed,callback);
                       $(elemento).show(speed,callback);
                       $(elemento).toggle(speed,callback);
                    });

O speed e o callback são opcionais

Os callback são funções que vão correr apenas quando o efeito finalizar.

Fade

                
                    $j(document).ready(function() {
                       $(elemento).fadeIn();
                       $(elemento).fadeOut();
                       $(elemento).fadeToggle();
                       $(elemento).fadeTo(); // usado para fazer fade à opacidade de 0 a 1
                    });

Sliding

                
                    $j(document).ready(function() {
                       $(elemento).slideDown());
                       $(elemento).slideUp());
                       $(elemento).slideToggle();
                    });
