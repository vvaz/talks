Git - Aula 3

Novo exercício

  • Criar um novo repositório no github.com para um livro de receitas
  • Adicionar a equipa Flag ao repositório
  • Clonar esse repositório para o ambiente local
  • Adicionar algumas receitas em ficheiros diferentes
  • Fazer commit e push main branch
  • Criar uma branch nova com modificações a cada receita
  • Fazer commit e push dessa branch
  • Alterar no github.com as receitas, adicionando fotografias das receitas no formato markdown
  • Fazer pull das alterações para o ambiente local.

Extra

  • Clonar um repositório de um colega, fazer uma branch nova e adicionar umas modificações a uma receita.
  • Fazer push.
  • Tentar criar conflitos nos repositórios dos colegas

git merge

O comando git merge é usado para juntar duas ou mais branches. O merge é um comando não destrutivo, ou seja, não altera o histórico das branches. 

            
              git merge branch

git revert

O comando git revert é usado para desfazer um commit. O revert é um comando não destrutivo, ou seja, não altera o histórico das branches.
