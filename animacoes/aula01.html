Existem duas formas principais de criar animações na web. Com CSS e com JavaScript. Já existiu uma terceira e ainda bem que já está extinta, alguém sabe qual é?
A forma com que escolhemos animar um elemento ou um grupo de elementos, depende das outras dependências do projecto, e que tipo de efeitos queremos gerar.
A recomendação que vos deixo, é usarem animações de CSS para transições algo simples e rápidas, como por exemplo, mudar estados de elementos, como botões com hover, ou menus que abrem e fecham.
Usar animações de JavaScript quando queremos ter animações mais avançadas como simular o movimento de uma bola com gravidade, o "parar", andar para trás ou até camera lenta.
Se escolhermos o JavaScript, podemos usar a Web Animations API ou uma library com as animações já pré-definidas.
